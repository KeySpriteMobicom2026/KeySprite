### 🎨 Interaction States Definition for Data Journey Map

#### 1. 节点（Node）交互状态
- Default:  
  Icon + Label in neutral gray, Status Badge outline only

#### 2. 视图切换按钮（View Toggle）
- Default:  
  Gray background, text in dark gray  
- Hover:  
  Slight shadow, text color darkens  
- Active/Selected:  
  Blue fill, white text, subtle border glow  
- Focus State:  
  Outline ring for accessibility  

🔘 Example: `[ Simplified View ] [ Full View ]`

---

#### 3. 控制面板按钮（Control Buttons）
- Default:  
  Outlined button style, neutral icon color  
- Hover:  
  Light background tint, icon brightens  
- Pressed/Active:  
  Icon moves down 1px, background darkens slightly  
- Disabled:  
  Icon fades to 40% opacity, no click response  

🔍 Example: `Zoom In` / `Expand All Steps`

#### 4. 流程路径（Flow Path）

  Thin line, light gray

  Entire path up to that step lights up in blue gradient  
- Branch Path Indicator:  
  Dashed line with semi-transparent fill  

🌀 Bonus: Scroll-driven activation — as user scrolls, path lights up step-by-step

#### 5. 小贴士提示框（Tooltip / Tip Box）
- Default:  
  Hidden  
- On Hover / Click:  
  Fade-in animation, slight bounce  
- Active:  
  Background highlight, pointer arrow aligns with trigger point
