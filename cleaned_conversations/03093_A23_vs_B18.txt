## 🔧 Design Process & Tools

CulturaLoom is not just a course — it's a , blending psychology-driven research with hands-on prototyping. The process follows an Emotion-First Design Framework, which includes:

### 1. Emotional Fieldwork 🎤🔍
- Conduct micro-interviews and diary studies using emotion-triggering prompts
- Use tools like Dscout or Lookback for real-time emotional feedback
- Map user emotions across time (e.g., weekly mood graphs + emoji logs)

### 2. Cross-Cultural Mood Modeling 🌏🧠
- Create  that reflect cultural display rules
- Use frameworks like Hofstede’s cultural dimensions or Trompenaars’ models to inform UI tone & interaction style
- Prototype with tools like Figma’s Localization Plugins or UXPressia for emotional storytelling

### 3. Adaptive UI Sketching 💡🖌️  
- Start with paper prototypes of emotion-responsive interfaces
- Build mood-aware wireframes (e.g., UI shifts from minimal to supportive based on user state)
- Tools: Figma, Adobe XD, Whimsical Emotion Maps

### 4. Micro-Interaction Prototyping 🌀⚡  
- Use platforms like ProtoPie or Principle to build dynamic emotional responses in real-time
- Example: a button that changes text + animation based on detected mood fatigue
- Bonus challenge: design a “calm mode” trigger (like a long press on the screen that softens the entire UI palette)

### 5. Empathetic A/B Testing 📊❤️  
- Test versions of your interface with and without emotion-cultural adaptation
- Measure engagement, perceived empathy, and emotional resonance
- Tools: Hotjar, Optimizely, UserTesting Emotional Feedback Tags

---

## 🎨 UI/UX Design Themes

To bring the emotional intelligence theme to life visually, here are some direction ideas we can explore:

### 🎨 Color System:
- Use a gradient-based emotion spectrum: cool blues/greys for calm → warm oranges/pinks for joy/energy
- Dynamic background tint based on user-reported mood (think: gentle shift between shades throughout the day)

### 🖼️ Visual Language:
- Emoji-inspired micro-illustrations to represent emotional states
- Soft blur transitions instead of sharp edges — mimicking the fluidity of feelings
- Floating visual anchors (like a pulsing sun/moon icon) to symbolize emotional cycles

### 🔄 Interaction Style:
- Mood-sensitive microcopy:  
  - If user seems low-energy: “Hey, you’ve been going hard lately. Want to take a slow path today?”  
  - If user is active: “You’re on fire today! Keep those creative sparks flying!”
- Emotional rhythm animations:  
  UI elements subtly pulse or sway in sync with user’s emotional tempo (detected via usage patterns or journal input)

## 🎬 UI/UX Tone & Voice Guide（初稿）

除了视觉风格和交互方式，我们还要教学生设计情绪语调（Emotional Tone）——这其实是很多设计师忽略的“第五层感官”。

比如：

| Emotional State | Microcopy Style | Visual Feedback | Cultural Sensitivity Tip |
|------------------|------------------|------------------|----------------------------|
| 😊 Joyful        | Lively, playful emojis, upbeat phrasing | Bouncy animations, vibrant colors | Avoid overstimulation in cultures that value restraint |
| 😔 Low Energy    | Gentle, supportive tone | Soft fades, minimal pop-ups | Offer “quiet mode” toggle for users who don’t want push interactions |
| 😠 Frustrated    | Direct but calming language | Subtle pauses before action steps | Use culturally neutral symbols to avoid triggering unintended offense |
| 🤔 Reflective     | Thoughtful questions, space to journal | Slower transitions, more whitespace | Include optional prompts for self-reflection without judgment |
